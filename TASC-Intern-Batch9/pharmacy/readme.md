🚀 Project: Pharmacy
Dự án này được phát triển theo từng giai đoạn rõ ràng, tập trung vào việc xây dựng một ứng dụng web hoàn chỉnh với các tính năng từ cơ bản đến nâng cao.
📅 Timeline & Giai Đoạn Phát Triển
Giai đoạn 1: Khởi Tạo Dự Án & Cấu Hình Cơ Bản (Ngày 1)
Thiết lập môi trường phát triển và cấu trúc dự án ban đầu.
✅ Cài đặt Java Development Kit (JDK) phiên bản 17.
✅ Cài đặt và cấu hình các thư viện dự án cần thiết (ví dụ: Spring Boot Starters).
✅ Thiết lập ánh xạ Entity với cơ sở dữ liệu (ORM).
✅ Tổ chức cấu trúc dự án với các package liên quan.
✅ Kết nối thành công với Cơ sở dữ liệu.
Giai đoạn 2: Tích Hợp Bảo Mật với Spring Security (Ngày 2)
Xây dựng nền tảng bảo mật vững chắc cho ứng dụng.
🔒 Cấu hình Security Filter Chain và Custom AuthenticationEntryPoint.
🔒 Bảo vệ và cấu hình các loại token (ví dụ: JWT) để xác thực.
Giai đoạn 3: Cấu Hình Xử Lý File & Hình Ảnh (Ngày 3)
Triển khai chức năng quản lý file và hình ảnh thông qua dịch vụ đám mây.
📤 Cấu hình chức năng upload và load file thông qua dịch vụ Cloudinary.
Giai đoạn 4: Xây Dựng Module Xác Thực Người Dùng (Ngày 4 - 5)
Phát triển các tính năng cốt lõi cho việc quản lý tài khoản người dùng.
👤 Xây dựng chức năng Đăng nhập (Login) và Đăng ký (Register).
🔑 Triển khai chức năng Quên mật khẩu (Forgot Password).
Giai đoạn 5: Quản Lý Người Dùng & Phân Quyền (Ngày 5 - 6)
Phát triển module quản lý người dùng với phân quyền và cấu hình thông tin cá nhân.
👥 Xây dựng module quản lý người dùng với chức năng phân quyền theo vai trò (role-based authorization).
⚙️ Cấu hình và quản lý thông tin cá nhân của người dùng.
Giai đoạn 6: Phát Triển Module Nội Dung & Sản Phẩm (Ngày 7 - 8)
Xây dựng các module chính liên quan đến nội dung và sản phẩm của ứng dụng.
🏷️ Xây dựng module Brand (Thương hiệu).
📚 Xây dựng module Category (Danh mục).
📝 Xây dựng module Blog.
🛍️ Xây dựng module Product (Sản phẩm).
Giai đoạn 7: Giỏ Hàng & Thanh Toán (Ngày 9 - 10)
Triển khai chức năng giỏ hàng và các phương thức thanh toán.
🛒 Xây dựng module Cart (Giỏ hàng).
💳 Tích hợp các cổng thanh toán như VNPAY và phương thức Thanh toán khi nhận hàng (COD).
📦 Xây dựng module Order (Đơn hàng).
